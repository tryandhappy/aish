# aish

# 概要
CLI SSH + AI (Claude, ChatGPT, Gemini等)

# 仕様
- 文字コード:UTF8 (UTF8MB4)
- 開発言語:Rust
- 対応OS:Windows, MacOS, Linux(Ubuntu)
- 対応AI:Claude, ChatGPT, Gemini等
- aish コマンドを実行するとSSHが起動し、AIが常にコンソールを監視する。
- /chatgpt と入力するとメインAIをChatGPTに切り替えることができる。
- /claude と入力するとメインAIをClaudeに切り替えることができる。
- /gemini と入力するとメインAIをGeminiに切り替えることができる。
- AIを切り替えたときにログインしていなければ、ログインする。
- ! マークを先頭に入れると、そのままコマンドを実行する。
- コマンドとその結果をAIが読み取り、いつでも助言できるようにする。
- /?と入力すると、現在のメインAIの解説を聞くことができる。
- /? <AI> と入力すると、そのAIの解説を聞くことができる。
- /? chatgptと入力するとChatGPTの解説を見ることができる。
- /? claudeと入力するとClaudeの解説を見ることができる。
- /? geminiと入力するとGeminiの解説を見ることができる。
- /? allと入力すると、すべてのAIの解説を見ることができる。
- ssh USER@HOSTでログインできるように。
- 内部的にsshコマンドを使用しても構わない。
- プロンプトを入力すると、AIが代わりにコマンド実行やサーバ内の調査を行うことができる。
- サーバ内で何らかのコマンド実行やファイル読み取りを行うときは必ずユーザに確認する。
